# Project BOM

This project demonstrates a simple browser layout with a topbar and an iframe filling the entire space.

## How to Run

1. Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
2. The page will display a topbar and an iframe below it.

## Project Structure

- `index.html`: Main HTML file containing the layout.
- `README.md`: Instructions for running the project.

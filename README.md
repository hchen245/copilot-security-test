# Project Overview

This project contains three distinct tasks, each demonstrating different functionalities and concepts in web development.

## Task 1: BOM (Browser Object Model)

This task demonstrates a simple browser layout with a topbar and an iframe filling the entire space.

### How to Run

1. Open the `tasks/task1/index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
2. The page will display a topbar and an iframe below it.

## Task 2: Calculator Files

This task is a feature-rich calculator implemented in HTML, CSS, and JavaScript. It supports both basic and scientific modes, along with additional features like memory storage and a history panel.

### Features

- **Basic Mode**: Displays only essential arithmetic buttons.
- **Scientific Mode**: Includes advanced functions like trigonometry, square root, and more.
- **Memory Functions**: Store, recall, add, subtract, and clear memory values.
- **History Panel**: Displays the last 5 clicked buttons, which can be clicked to repeat actions.
- **Dark/Light Theme**: Toggle between dark and light themes.

### How to Run

1. Open the `tasks/task2/index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
2. Use the calculator interface to perform calculations.

## Task 3: Canvas

This task demonstrates the use of the HTML5 Canvas API to create interactive or graphical content.

### How to Run

1. Open the `tasks/task3/index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
2. Interact with the canvas to explore its features.

## Purpose

This project was completed entirely using GitHub Copilot in Visual Studio Code. The purpose of this project is to test the correctness and effectiveness of AI-generated work results after assigning different tasks to the AI.

## Project Structure

- `tasks/task1/`: Contains files for the BOM demonstration.
- `tasks/task2/`: Contains files for the calculator project.
- `tasks/task3/`: Contains files for the Canvas demonstration.
- `README.md`: Instructions for running the project.
